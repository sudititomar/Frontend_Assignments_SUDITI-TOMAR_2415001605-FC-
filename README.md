13-week hands-on Full Stack Web Development lab manual covering HTML5, CSS3 Flexbox, JavaScript ES6+, jQuery, Bootstrap, and React.js with Router.

### Weeks 1-3: Frontend Foundations

- **Event Delegation** (Image gallery with thumbnails)
- **Async JS** (Promises, `setTimeout()` simulated API)
- **jQuery** (FAQ toggle, fade effects, dynamic CSS)
- **Advanced DOM** (Chessboard generator, enhanced To-Do with up/down/delete)
- **Fetch API** (Real universities API by country)

### Weeks 11-13: Modern React
- **React Setup** (Vite/CRA, JSX, Components)
- **Props & State** (Reusable Card component, controlled forms with `useState`)
- **React Router** (Multi-page app: Home/About/Blog)
